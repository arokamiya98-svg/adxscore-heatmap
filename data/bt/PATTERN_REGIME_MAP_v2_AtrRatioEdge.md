## ★ 納品物：ケルトナー点灯条件に使える優位ATR Ratioレンジ

**定義（再現可能）**: `Ratio = 現在の iATR(H1,16) ÷ 直近960本H1バーの iATR(H1,16) 中央値`（960 = 8週×5日×24h）。インジ側で960本ローリング中央値を持てば完全再現可。

1. **BUY点灯帯 = 0.70 ≤ Ratio < 1.00**（押し目/圧縮帯）: dedup N=103 / WR53.4% / **PF1.88**（生947でも N=245 / WR47.8% / PF1.41）。下半分0.70-0.85が最強(PF2.0)。`H1_ATR_Pair>1`(短ATR拡張開始)併用で PF2.24(N=49)。
2. **SELL点灯帯 = 1.00 ≤ Ratio < 1.20 かつ D1がDN局面(DI-優勢)**: dedup N=21 / WR52.4% / **PF2.45**（生947でも N=41 / WR48.8% / PF2.41 = 頑健）。**同帯でもUP局面はPF0.70で無効＝局面ゲート必須**。
3. **共通OFF = Ratio ≥ 1.20 の SELL は死亡**（dedup PF0.41 / Net-$257、生でもPF0.74）。ただし**BUYは≥1.20でも生存(PF1.3〜1.5)** なので「高Ratio=消灯」はSELL限定で、方向非依存の一律ゲートにはできない。
4. **適用注意**: BT局面はUP80.3%/DN19.7%偏り。BUY帯は主にUP相場で計測 → 現DN局面では割引いて見る。逆にSELL-DN帯は今と同じDN局面での計測 → 現局面に適用しやすいがN=21と小。

## 0. 入力データ・前提

- ファイル: `data/bt/ATR_WidthSignal_BT_h4adx46.csv`（UTF-16LE, 88列, **raw 947件 → 6h dedup 391件**）
- 期間: 2024-01-02〜2026-06-01
- 全件 SL=ATR_Avg32×2.0 / **TP/SL比=きっかり1.600（全件固定RR）** / Result は WIN(419)/LOSS(528) のみ（建値なし）

### 方法論的確認
固定RR1:1.6なので **WIN=+1.6R / LOSS=-1.0R**。全体 PF1.36 ≈ impliedPF1.35 → **WRが最もクリーンなエッジ指標**。PFとimpPFが乖離するセルは「勝ち時のピップ幅>負け時」のピップ規模ノイズ（局面依存の単発大相場）であり過大評価しない。表に impPF（カウントベース）を併記。

### 分布（dedup 391）
`min=0.491 / P10=0.795 / P25=0.898 / P50=1.038 / P75=1.171 / P90=1.311 / max=1.778`
ビン件数: `<0.70=10 / 0.70-1.00=148 / 1.00-1.20=150 / 1.20-1.40=82 / 1.40-1.70=0 / 1.70+=1`
→ **1.40超(HIGH帯)はほぼ空**。**実用レンジは0.70〜1.40**。

## 8. 構造的解釈（なぜ効く/効かないか）
- **BUY=圧縮の解放を買う**: Ratio<1.0（ATRが8週中央値以下に沈む）は「ボラ圧縮＝エネルギー溜め」。そこに短ATRが立ち上がる(Pair>1, RISING_DECEL)とボトムアウト初動を捕える。Ratio1.2超(=ボラ放出済み)で買うと高値掴みでエッジ薄れる。XAU「買いは押し目」の数値的肖像。
- **SELL=拡張の天井を、ただし局面が下のときだけ売る**: SELLが効くのはRatio1.0-1.2（中央値をやや超えた拡張帯）で**かつD1がDN**のとき。圧縮帯(<1.0)のSELLは方向根拠が弱く拮抗、拡張ピーク(>1.2)のSELLは反転確定せず損切り。「売りは拡張」は**局面(DN)条件付き**でのみ成立。
- **非対称の本質**: BUYは「水準が低い」が条件、SELLは「水準がやや高い×局面が下」が条件。だから単一しきい値の一律ゲートにできず、方向別の窓になる。

## 9. 留意点
- **DN局面サンプル不足**: 全体DN=77件(19.7%)。SELL-DN帯N=21（参考超えだが小）。フォワード再確認推奨。
- **現局面への適用注意**: BTはUP80%。BUY押し目帯は主にUP相場の計測値 → 現DN局面では割引く（BUY×DN N8もPF2.67で構造はロバスト寄り）。SELL-DN帯は現局面と同質サンプルだが小N。
- **dedup差**: BUY帯dedup PF1.88は「6h窓の初動fire」値。常時点灯インジでは生PF1.41寄りが保守線 → **保守1.4〜初動1.9**で読む。
- **pips平均は帯間比較不可**: SL=ATR×2でピップ幅がATR水準に比例。エッジ比較はWR/PFで。
- **>1.40帯は実質データ無し** → 実効的なSELL上側OFFは1.20。

## 10. 再現性スペック（ケルトナー実装に渡す核）
```
H1_ATR_Ratio_Median = iATR(H1,16)[現在] / Median{ iATR(H1,16) の直近960本 }
  960本 = ATR_Median_Weeks(8) × 5 × 24
  CalcMedian: ATR>0の値のみ・有効10本未満は無効・偶数時は上側中央値(tmp[cnt/2])
H1_ATR_Pair = iATR(H1,16) / iATR(H1,32)   // >1.0で短ATR>長ATR(拡張側)
H4_ATR_Ratio_Median = iATR(H4,8) / Median{ iATR(H4,8) の直近240本 }  // 240=8×5×6
Zone: <0.70=LOW / 0.70-1.40=NORMAL / >1.40=HIGH（参考。点灯帯は上記の細帯で）
```
