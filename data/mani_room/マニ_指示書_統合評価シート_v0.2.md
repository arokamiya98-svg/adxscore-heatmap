## 3. v2 仕様

### 3.1 ドリルダウン: トレード日はシグナルカードの下にトレードカード追加

```
─── シグナルカード（既存のまま）───
[PatE SELL] 20:00 JST (server 14:00) @ 4470.14  pass ✅
  MFE/MAE: 12h xx/xx → 24h xx/xx → 36h xx/xx → 48h xx/xx

─── 🔴 実トレード ───（トレードがあった日のみ表示）
[SELL] 14:32 → 18:05 JST   -40.0 pips   デイトレ ★3
  タグ: #凪離脱
  MFE/MAE: 12h xx/xx → 24h xx/xx → 36h xx/xx → 48h xx/xx   ← h1_mfe_usd_12h〜48h
  新規理由: （折りたたみ。クリックで展開、デフォルト閉）
```

**核心: シグナルカードとトレードカードの MFE/MAE 推移を同一フォーマットで並べる。** 「シグナルの伸び方」と「自分の執行の置かれた環境」が同じ言語で対比できる。

#### トレードカードに出すもの（これだけ。精査済み）
- 方向（▲▼、シグナルと同じ形言語）/ entry_jst〜exit_jst 時刻 / pips（符号で勝敗がわかる）
- スタイル / ★（時間帯ラベル）/ タグ
- h1 MFE/MAE 12/24/36/48h 推移
- 新規理由（折りたたみ、デフォルト閉。認知負荷対策）

#### 出さないもの（禁止）
- 損益額・ロット・累計系すべて
- 評価コメント・反省・考察の常時表示（反省/考察はカード外。出すなら折りたたみ内のみ）
- 勝率・集計パネルの類
