## 3. カレンダーUI仕様

### 3.1 月別カレンダーグリッド（trades_calendar と同型）

- 期間: 2025-03 〜 最新（CSVから自動）
- **日付は `date` 列（JST基準）**。time_jst で表示、ドリルダウンには time_server を併記（チャート照合用、「サーバー時間=チャート表示時間」と注記）

### 3.2 日セルの発火表示

- その日の発火をパターン色のドットで表示（複数発火 = 複数ドット、多い日は「+n」集約可）
- **パターン色は v4 実機の矢印色と完全一致させる**（認識の一貫性）:

| パターン | BUY | SELL |
|---|---|---|
| PatA | Gold (#FFD700) | Goldenrod (#DAA520) |
| PatB | Aqua (#00FFFF) | DeepSkyBlue (#00BFFF) |
| PatC | LimeGreen (#32CD32) | SeaGreen (#2E8B57) |
| PatD | Magenta (#FF00FF) | MediumVioletRed (#C71585) |
| PatE | Orange (#FFA500) | DarkOrange (#FF8C00) |

- **pass_all=FALSE（フィルター抑制発火）は薄表示**（opacity 0.3 程度）。「実機なら見えなかった発火」が見える化される — これがフィルター理解の核心
- BUY/SELL の区別: ドットの形 or 上下位置で（▲▼等）。**色は方向に使わない**（色=パターン、これも v4 と同じ言語）

### 3.3 ドリルダウン（日クリック）

その日の発火カードを表示:

```
[PatE SELL] 20:00 JST (server 14:00) @ 4470.14   pass_all: ✅
  環境: ATR Zone=NORMAL / H1 ADX=xx / cross_dir=xx / H4 phase=xx
  フィルター: 9本中ヒットなし（or「F1, F7 にヒット → 実機非表示」と明示）
  MFE/MAE推移: 12h: xx/xx → 24h: xx/xx → 36h: xx/xx → 48h: xx/xx
```

- MFE/MAE 推移は小さな棒 or ステップ表示で「伸び方の形」が見えると良い（マニ裁量）
- 環境値の表示は**段階ラベル優先**（ADXは zone 列を使う等）。連続値の生表示より段階認識（あろさんの認識粒度は 20/25/30/35 の閾値段階）

### 3.4 実トレード ON/OFF オーバーレイ（執行確認用）

- ヘッダーにトグルボタン「実トレード表示 ON/OFF」（デフォルト OFF）
- ON 時: 実トレードがあった日のセルに枠線 + エントリー方向マーク（trades_enriched_full の 約定日・direction を使用）
- **表示はそれだけ**。損益・評価・反省は出さない（執行確認 = 「シグナルが出た日、ポジションを持ったか」だけ見る）

### 3.5 フィルター切替（上部）

- 円グラフ等は不要。シンプルに:
  - パターン別 表示切替（PatA〜E チェックボックス）
  - 「pass_all のみ / 全発火」切替
  - BUY / SELL 切替
