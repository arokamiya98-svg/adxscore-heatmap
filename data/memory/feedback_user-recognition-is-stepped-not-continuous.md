あろさんのトレード判断・認識は**連続値ではなく段階的な閾値**で動く。ADX なら 20以上 / 25以上 / 30以上 / 35以上 のような閾値ベース。細かい連続値（ADX 22 vs 23 など）の差で判断していない。UIも連続グラデーションでなく段階化すべき。

**Why:** あろさん明示「22とかで判断してるわけじゃない、20以上25以上30以上とかで明度変えるなりした感じがいい」（2026-06-10、日次研究カレンダー実物確認時）。連続グラデーション表現は認識粒度と乖離する。

**How to apply:**
- ADX/DI/その他指標を UI 表現するとき、連続値マッピング（線形 normalize→グラデ）を第一選択にしない
- 認識粒度（閾値）に合わせた**段階的明度**を優先
- ただし「HOT/弱/中/強」のような評価ラベルは出さない（[[research-purpose-and-rules]] に基づく、エントリーロジック混入回避）。**段階化は明度・色相のみで、テキストラベルは出さない**
- 段階の刻み方は実測経験ベース（XAUUSD なら ADX 15/20/25/30/35 が候補）
- 連続グラデーションが適する場面もある（D1 帯のような大局背景は連続不要、5パターン色分けで足りる例あり）
- 関連: [[anti-perfectionism-true-meaning]]（粒度の使い分け）, [[research-purpose-and-rules]]
