2026-06-16、WaveLog_Export_v16.csv（H1波形、手描きトレンドラインのBU/PD）の統計解析。あろさんが「すごい法則」「H1で見れる波」と反応した発見。

**手描きBU/PD波の数値的肖像（全期間2025.01-2026.06, VEL+ATR合算 BU145/PD120）**:
- **BU = 短い・大きい・ATRが開く**: dur中央17bar / 値幅中央336p / **ATR拡張 +25.6%** / MFE434
- **PD = 長い・小さい・ATRが閉じる**: dur中央23bar / 値幅240p / **ATR拡張 −11.2%** / MFE333
- PD/BU duration比 ≈ 1.35〜1.47（[[trading-method-summary]]の黄金比階段φ=1.618に符合）
- dur平均(BU32/PD38)は中央値より大きい＝右に長い裾。代表値は中央値で見る

**BUとPDは「別の動物」が符号レベルで確定**: ATR拡張%がBUプラス/PDマイナスで真逆。BU=ATRボトムアウトからの拡張ゾーン（獲りにいく）、PD=収縮ゾーン（待つ/逆らわない）。あろさんの手法（ATRボトムアウト狙い＝BU波）の数値的裏付け。

**局面バイアス検証（全期間 vs 2026.03以降の下落もみ合い, [[bt-regime-bias]]）**:
- **質的構造は局面を超えて不変**: BU拡張(+)/PD収縮(−)の符号は直近でも完全保持（BU+35.9%/PD−16.8%）＝構造は本物、局面バイアスでない
- **BUのdurationは鉄板**(17→18bar)。伸縮するのはPD(23→31bar)。「獲りにいくBUは一定、待つPDが状況で変わる」
- PD/BU比は直近1.72（全期間1.35）＝直近の方がφに近い
- **値幅は約2倍にスケール**(BU336→660, PD240→463)＝前セッションの「PD末期→ボラ解放started」が数字に出た。質は同じ・量だけ膨張

**How to apply**: BU/PD法則は構造として信頼可（局面超え）。ただし値幅の絶対値は局面のボラで補正（今はボラ解放で約2倍）。BU dur17-18が最も収束する核。これはフェーズ2「感覚のロジック化」([[roadmap-sensory-to-logic-phase]])の確かな足場＝手描き感覚が普遍構造を捉えていた実証。
