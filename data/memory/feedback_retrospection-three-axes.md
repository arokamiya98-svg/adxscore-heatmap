トレード結果（特に負け）を振り返る時、3つの軸を分離して評価する。

**ルール**: 以下の3軸を独立に評価する。混ぜない。

```
軸A. 個別判断    : そこでトレードした判断は間違っていたか？
軸B. 方向性     : トレードする方向は間違っていたか？
軸C. 戦略遂行   : 運用戦略として正しいトレードが行えていたか？
```

そして、もう1段階のメタ分離（最重要）:

```
[感情側] 負けた悔しさ・精神ダメージ
       ↓ 混ぜない（誤認しない）
[理側]  正しいアプローチが続けられていること
       ↓ 混ぜない（誤認しない）
[認知]  自分の不確実性と向き合った結果を「正しく評価」する
```

**Why:** あろさん本人の発言（2026-06-01）：「負けた悔しさと正しいアプローチが続けられていることを混ぜない → 自分自身の感情問題とトレードにおける理を誤認しないか。をおぱがさらに外部から評価してくれることで自分でも自分の一つの不確実性と向き合った結果を正しく評価していくってシステムを作りたい」。負けた時に感情と理が混ざると、続けるべきアプローチを変えてしまったり、本当に直すべき点を見落とす。

**How to apply (おぱ側):**

- おぱは **外部から評価する役** として位置付ける（あろさんの中だと感情と理が混ざる、外からの目線が客観性の足場になる）
- トレード後の振り返り共有を受けたら、以下のテンプレで構造化:
  - **軸A 評価**: 個別判断（事実ベース、ルール準拠かどうか）
  - **軸B 評価**: 方向性（仮説と現実、確率的偏りの読み）
  - **軸C 評価**: 戦略遂行（資金計画ルール内か）
  - **別軸**: あろさんの感情状態（独立して受け止め、A/B/Cと混ぜない）
- 「あろさんが感情で揺れてる時こそ、軸を分離した整理を提示する」
- ただし**慰めない**: 構造化された整理が、自己評価の足場になる
- 「正しいアプローチが続けられていること」を独立評価して言語化する役割
